# ChangeLog ☯️

> **"The only constant is change." / 「唯有變動，才是永恆。」**
> 
> A modern I Ching (Book of Changes) journaling and learning app.  
> 記錄生命的每一次變動。一個結合現代化 UI/UX 介面，專注於《易經》占卜、記錄與學習的跨平台應用程式。

---

## 🌟 Features / 核心特色

* **Divination & Ritual (占卜與儀式感):** An intuitive divination process with a modern aesthetic. / 提供直覺且具現代美感的起卦流程。
* **The Change Log (變化日誌):** Systematically track your hexagrams, contexts, and reflections. / 系統化記錄每一次起卦的背景、卦象與後續驗證。
* **Wisdom Hub (智慧學習):** In-depth analysis of the 64 hexagrams to master I Ching philosophy. / 內建完整 64 卦解析，協助深度理解易理。
* **Data Sovereignty (資料自主):** Local-first storage with CSV/JSON export and encrypted cloud backup. / 預設本地儲存，支援 CSV/JSON 匯出與加密雲端備份。

## 🛠 Tech Stack / 技術棧

* **Framework:** Flutter (iOS & Android)
* **Database:** Isar (High-performance NoSQL Object Database)
* **State Management:** Riverpod / Provider
* **Localization:** flutter_localizations (Support for English & Traditional Chinese)
